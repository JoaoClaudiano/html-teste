Condições para utilização do CodeTest

1. Aceitação

Ao acessar e utilizar o CodeTest, você concorda com estes Termos de Uso. Caso não concorde com alguma condição, recomenda-se não utilizar a ferramenta.

2. Descrição do serviço

O CodeTest é uma ferramenta de caráter educativo e informativo para escrita, visualização e análise preliminar de código HTML, CSS e JavaScript, executada inteiramente no navegador do usuário. Não há cadastro, planos pagos ou processamento em servidor.

3. Responsabilidades do usuário

  • Não utilizar a ferramenta para executar, testar ou distribuir código malicioso, destrutivo ou ilegal.
  • Não tentar sobrecarregar, explorar ou comprometer a disponibilidade do serviço.
  • Reconhecer que o código inserido é de sua exclusiva responsabilidade.
  • Não utilizar o serviço para fins que violem legislação aplicável.

4. Propriedade intelectual

O código-fonte, design e conteúdo do CodeTest são de propriedade do autor. O código inserido pelo usuário no editor permanece de responsabilidade e propriedade exclusiva do próprio usuário.

5. Isenção de responsabilidade

O CodeTest é uma ferramenta educacional e não substitui validadores oficiais, auditorias de segurança ou testes profissionais. Os resultados apresentados não garantem conformidade total com padrões W3C ou requisitos legais. Para validação oficial, utilize o W3C Validator.

6. Processamento de dados

Todo o processamento ocorre exclusivamente no navegador do usuário. Nenhum dado digitado no editor é enviado a servidores externos. Consulte a Política de Privacidade para informações sobre coleta anônima de dados de acesso.

7. Alterações

Estes Termos de Uso podem ser atualizados a qualquer momento, sem aviso prévio. Recomenda-se a revisão periódica desta página.
