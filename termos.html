2. Descrição do Serviço

O CodeTest é uma ferramenta de caráter educativo e informativo para escrita, visualização e análise preliminar de código HTML, CSS e JavaScript. A ferramenta é executada inteiramente no navegador do usuário (client-side), sem cadastro, sem planos pagos e sem processamento em servidor próprio.

O serviço inclui:

  • Editor de código com syntax highlighting (HTML, CSS e JavaScript)
  • Pré-visualização em tempo real com simulação de dispositivos
  • Análise automatizada de qualidade, acessibilidade e SEO (17 pontos)
  • Compartilhamento de código via URL comprimida
  • Exportação do código como arquivo .html
  • Histórico local das últimas análises realizadas

O autor reserva-se o direito de modificar, suspender ou encerrar qualquer funcionalidade a qualquer momento, sem aviso prévio ou obrigação de compensação.
