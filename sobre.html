O que é o CodeTest?

O CodeTest é uma ferramenta gratuita para escrever, visualizar e validar código HTML, CSS e JavaScript diretamente no navegador — sem instalar nada, sem criar conta, sem enviar dados a servidores.

Todo o processamento acontece localmente no seu dispositivo. O código que você digita nunca sai do seu navegador, garantindo total privacidade e funcionamento offline parcial.

⚡

Funcionalidades principais

✏️ Editor avançado

Abas separadas para HTML, CSS e JS com syntax highlighting via CodeMirror.

📱 Pré-visualização responsiva

Simule mobile, tablet e desktop em tempo real sem sair do editor.

🔎 Análise automática

Verificação de acessibilidade, SEO e boas práticas com pontuação de 0 a 100.

🔗 Compartilhamento por URL

Comprima e compartilhe seu código inteiro em um único link via LZ-String.

💾 Exportação e histórico

Exporte como .html e acesse o histórico das últimas 10 análises.

🌙 Modo escuro

Tema persistente via localStorage, com detecção automática da preferência do sistema.

📋 Colar da área de transferência

Cole código existente diretamente no editor com um clique.

🏅 Validação no W3C

Botão para enviar o código ao validador oficial do W3C com um clique.

O motor de análise

A análise usa DOMParser nativo do navegador para processar o HTML como um documento real — sem expressões regulares frágeis. São verificados 17 pontos, incluindo:

  • DOCTYPE, charset e viewport presentes
  • Título (presença e comprimento ideal de 30–60 caracteres)
  • Meta description (presença e comprimento ideal até 160 caracteres)
  • Imagens sem atributo alt (acessibilidade)
  • Links externos sem rel="noopener" (segurança)
  • Hierarquia de headings e múltiplos <h1>
  • Campos de formulário sem <label>
  • Tags obsoletas (font, center, marquee…)
  • Atributo lang na tag <html>
  • Open Graph e favicon detectados

Tecnologias utilizadas

O CodeTest é construído inteiramente com tecnologias abertas e sem dependências de servidor:

  • HTML5 DOMParser
  • CodeMirror 5
  • LZ-String
  • CSS3 custom properties
  • Vanilla JavaScript
  • Google Fonts (Inter)
  • Google Analytics 4
  • LocalStorage API
  • Clipboard API
🔒

Privacidade e dados

Nenhum dado pessoal é coletado. O código inserido no editor nunca é enviado a servidores. O único armazenamento utilizado é o localStorage do seu próprio navegador, para salvar o tema preferido e o histórico de análises recentes.

Você pode apagar esses dados a qualquer momento limpando os dados do site nas configurações do navegador. Consulte a Política de Privacidade completa para mais detalhes.

🚀

Projeto independente

O CodeTest é um projeto independente de código aberto, mantido por um desenvolvedor indie. Não está vinculado a nenhuma empresa ou organização. Sugestões, relatos de bugs e feedbacks são muito bem-vindos!

Para validações formais, recomenda-se o Validador Oficial do W3C. O botão "Validar no W3C" na ferramenta facilita esse processo.
